# Бот для обучения автотестированию

Telegram-бот, который помогает изучать автотестирование.

## Запуск

1. Установите Java 17+
2. Склонируйте репозиторий
3. Установите переменную окружения `BOT_TOKEN` с вашим токеном от BotFather
4. Запустите: `./mvnw spring-boot:run` (или соберите JAR)

## Пример запуска в терминале

```bash
export BOT_TOKEN=ваш_токен
java -jar bot.jar
